# Simple POKEDEX JS App

---

## Introduction

---

This is a simple Pokedex app.It shows a list of 150 Pokemons loaded from external public API : [Pokemon API](https://pokeapi.co/api/v2/pokemon/?limit=150).

---

## Technologies used

---

-   `HTML`
-   `CSS`
-   `Javascript`
-   `Bootstrap v4.3.1`
-   'jQuery 3.6.0'
